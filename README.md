# SixJax Poker

6-card, point-based poker game for 2-6 players. Built with Flask + vanilla JS.

## Run Locally

```bash
pip install -r requirements.txt
python server.py
```

Open http://localhost:8080

## Deploy to Railway

1. Push to GitHub
2. Connect repo in Railway dashboard
3. Railway auto-detects Python + deploys

## Custom Domain

Add your domain in Railway settings and point DNS (CNAME) to the Railway URL.
